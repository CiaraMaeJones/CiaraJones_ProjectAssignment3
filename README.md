# CiaraJones_ProjectAssignment3
Cleaning and preparation of a GBIF Lepidoptera occurrence dataset for spatial and temporal analysis in ESCI 620 Big Data in Environmental Science.
This repository contains the cleaned dataset and cleaning log for my ESCI 620 Week 3 course project assignment.

Files:
- LEPIDOPTERA_CLEANED_2026-04-05.csv
- LEPIDOPTERA_CLEANING_LOG_2026-04-05.pdf

The dataset was cleaned in R from a GBIF Darwin Core Archive and prepared for spatial and temporal analysis of Lepidoptera occurrences in Ohio.
